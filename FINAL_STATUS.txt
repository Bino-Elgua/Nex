API ENDPOINTS:

GET /health
  Returns: {"ok": true}
  Purpose: Health check

GET /status
  Returns: {
    "gateway": "Nex v1.0",
    "uptime": <ms>,
    "executions": <count>,
    "sessions": <count>,
    "timestamp": <ms>
  }
  Purpose: Gateway statistics

POST /execute
  Body: {"graph": <NexGraph JSON>}
  Returns: {
    "status": "success|error",
    "result": <execution result>,
    "executionMs": <time>,
    "timestamp": <ms>,
    "error": <message if status=error>
  }
  Purpose: Execute a graph and return results
