## Known Limitations

### Scope (By Design)

1. **Single-machine state**
   - Graphs executed locally
   - No distributed coordination
   - Fix in v2.0 with persistence

2. **In-memory execution**
   - No graph persistence
   - No execution history
   - Fix in v2.0 with database

3. **No external I/O**
   - HTTP calls are stubs in Tier 3
   - No actual network calls
   - Fix in v1.1 with real HTTP client

4. **No authentication**
   - Designed for internal use
   - No API keys or auth tokens
   - Fix in v1.1 with OAuth/JWT

### Workarounds (If Needed)

- **Persistence**: Externally save/load graphs
- **Distribution**: Run multiple gateway instances + load balancer
- **Auth**: Front with reverse proxy (Nginx with auth module)
- **Real HTTP**: Implement http/get in Tier 3 with actual fetch
